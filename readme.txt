Allocate new users a role for each site in your multisite installation.

== Description ==

Running a Multisite WordPress?

You no longer need to be manually add new users to each site. 

With this plugin, when a new user activates their account, they will be assigned a role for each of your sites.

You determine each role on a per site basis under the Super Admin | Options menu.

== Installation ==

Upload the plugin to your site, activate it, then visit "Super Admin | Options menu" to assign default roles to each of your sites.

== Frequently Asked Questions ==

= Does this require a multisite installation? =

Yes, WordPress takes care of default roles on non-multisite installations.
